# Contributing

Thank you for your interest in contributing to RAZOR Auto Labeler.

## Reporting Bugs
Please open an Issue and include:
- Operating system
- Python version
- Steps to reproduce
- Screenshots if available

## Feature Requests
Feature requests are welcome. Please describe your idea clearly.

## Pull Requests
1. Fork the repository.
2. Create a new branch.
3. Make your changes.
4. Submit a Pull Request.

Please keep code clean and well documented.
